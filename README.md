# wireshark-network-analysis lab

## Objective
Capture and analyze network traffic to understand protocols and communication.

## Tools Used
- Wireshark

## Steps
1. Installed Wireshark
2. Captured live network traffic
3. Applied filters (HTTP, DNS, TCP)

## Findings
- Observed DNS queries resolving domain names
- Identified HTTP vs HTTPS traffic
- Analyzed packet structures and IP addresses

## Key Learning
- DNS translates domain names to IPs
- HTTPS encrypts traffic unlike HTTP
